## Usage Guide
### Contract Deployment
Deploy `InterplanetaryEconomyContract` via Soroban CLI and invoke methods like `register_entity` or `initiate_trade`.

### CLI Commands
- Register planet: `python cli/hyper_ai_brain_cli.py register-planet --name Earth --resources 1000000`
- Allocate resources: `python cli/hyper_ai_brain_cli.py allocate-resources --planet Earth --need 50000`
- Initiate trade: `python cli/hyper_ai_brain_cli.py initiate-trade --from-planet Earth --to-planet Mars --amount 1000`
- Prevent crime: `python cli/hyper_ai_brain_cli.py prevent-crime --activity gambling`
- View data: `python cli/hyper_ai_brain_cli.py show-data`
